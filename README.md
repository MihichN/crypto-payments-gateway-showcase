# Crypto Payments Gateway Showcase

Public-safe showcase of a crypto payment platform with landing page, merchant dashboard, checkout page, payment API, and wallet prototype.

## Highlights

- Merchant dashboard for invoices, settings, and payment analytics.
- Checkout page for crypto payments and status tracking.
- Payment API for invoice lifecycle and frontend integration.
- Wallet prototype for balance/address-oriented user flows.
- Multi-service architecture with clear frontend/backend boundaries.

## Tech Stack

- Next.js, React, TypeScript, Tailwind CSS.
- Node.js, Express, MySQL.
- Blockchain-related libraries in the private codebase.
- QR code generation and merchant-facing APIs.

## Repository Contents

- `docs/architecture.md` - high-level payment system design.
- `examples/payment-status/paymentStatus.ts` - safe invoice status reducer.

## Confidentiality Notice

The production implementation is private. This showcase excludes private keys, seed phrases, RPC keys, merchant secrets, webhook signing logic, blockchain adapters, and production payment flows.
